# Results Guide

This directory contains generated analysis outputs. Most tables, figures, and
RDS files are ignored by git because they can be large and are regenerated by
the numbered scripts.

The rendered report is committed at:

```text
docs/index.html
```

GitHub's repository file browser shows that file as HTML source. The rendered
webpage is served through GitHub Pages after the report is merged into `main`
and Pages is enabled from the `/docs` folder:

```text
https://parthdoshi97.github.io/Rna-seq_DE_gene_CO_expression/
```

## Current Result Highlights

- 18,158 genes were tested for differential expression.
- 4,823 genes passed `padj < 0.05` and `abs(log2FC) > 1`.
- WGCNA used 5,000 high-variance genes and identified 12 non-grey modules.
- The turquoise module is strongly ER-associated and enriched for chromosome
  segregation and cell-cycle biology.
- The brown module is enriched for lymphocyte activation and adaptive immune
  response.
- The greenyellow module is preserved but has a strong synaptic signaling
  annotation, so it is prioritized by the enrichment gate rather than by
  tumor-specific rewiring.
